# Archology

Simple project for self-hosting MCP server to expose integrations to AI tools.

![Arcology](assets/arcology.jpg)

## Tools
- Cursor AI
- ChatGPT5
- Codex

## Integrations
- Obsidian

## Workflow

```
ChatGPT / Cursor >> MCP over HTTPS >> ngrok tunnel >> Archology MCP Bridge >> Obsidian REST API
```
